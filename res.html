Learning

Khan Academy

A popular learning resource that is completely free. They offer courses on a variety of topics in all of the major disciplines. Their courses in computer programming offer a great foundation by covering the basics of JavaScript, HTML, and CSS.

Tool

Scratch

A visual block-based programming language for creating animations, games, and interactive stories. This is a great tool for younger children. Developed and maintained by the Lifelong Kindergarten Group at the MIT Media Lab.

Learning

Codecademy

An excellent learning resource that offers free and paid options. Their course materials range from beginner to advanced, and they have a large and supportive community that is willing to help newer programmers.

Learning

CodeClub

An initiative that promotes the formation of local after-school code clubs. It features fun lessons in a range of programming languages. The example projects are straightforward with excellent tips and images.

Tool

Tynker

A visual programming solution geared towards younger coders. Their program is recommended by many of the top brands in the industry, and they currently boast 60 million users. Projects include Minecraft modding, app creation, and robotics.

App

Swift Playgrounds

This application is phenomenal for teaching the basics of code to absolute beginners. The built in curriculum follows a clear path that covers many programming fundamentals.

Learning

SoloLearn

A community platform for learning to code in multiple languages. It is universally available on all platforms, and includes curriculum for over 12 languages. It utilizes badges, student-to-student mentoring, and peer competitions to motivate learners.

Social

Stack Overflow

Commonly used by developers of all skill levels to solve problems related to virtually every programming language. Users can learn from mistakes others have made, or participate by assisting others in overcoming roadblocks.
