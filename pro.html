Learning Projects

Below is a selection of projects from around the web that are related to three beginner friendly programming languages. Scratch, Ruby, and Python were selected because they are visually appealing, easy to decipher, and they are featured in STEM programs such as CoderZ. The projects scale in difficulty with Project 1 being the easiest to complete and Project 3 being the most difficult.

5 Best Programming Languages for Kids. (2017, February 16). Retrieved from https://gocoderz.com/blog/5-best-programming-languages-kids/

Scratch Projects

The following Project is hosted by Raspberry Pi. It is part of the Raspberry Pi Digital Making Curriculum.

Project 1
Rock Band

Learn how to code musical instruments in Scratch. In this project learners will use basic programming constructs to create a simple program.

Ruby Projects

This game based learning project was originally developed by Ryan Bates, but it has been adapted by Bloc for the web.

Project 2
Ruby Warrior

Learn about the Ruby language in this triumphant quest of adventure, love and destiny. Learners will discover Ruby basics such as methods.

Python Projects

The following projects are hosted and developed by Codecademy. Python is a great first language as it is easy to read and versatile.

Project 3
Python Syntax

This introductory lesson teaches programming fundamentals using Python. Learners will become familiar with Python syntax.
